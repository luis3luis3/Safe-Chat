Willkommen bei Safe Chat!

Über:

Safe Chat ist eine Chat-Anwendung, die für maximale Sicherheit entwickelt wurde. In jedem Fall werden nur 20 Nachrichten pro Chat in der Datenbank gespeichert. Wenn eine wietere Nachricht gesendet wird, wird die 21te Nachricht gelöscht. Benutzer haben auch selbst die Möglichkeit, den Inhalt eines jeden Chat vollständig zu löschen. Dies kann durch Klicken auf die Schaltfläche "Papierkorb" oben links auf der Chat-Seite erreicht werden. Chatroom-Namen müssen außerhalb dieser Anwendung der Personen übermittelt werden, wodurch eine weitere Sicherheitsstufe hinzugefügt wird.

Nutzung:

Auf der Anmeldeseite können Sie den gewünschten Chatraum und den gewünschten Namen eingeben. Wenn der Chatroom, den Sie eingeben möchten, bereits erstellt wurde, können sie alle gespeicherten Nachrichten in diesem Chatroom lesen. Wenn der Chatroom noch nicht Existiert wird der Chatroom neu erstellt. Nach dem Login im Chatroom können Nachrichten durch Eingabe in das Textfeld gesendet werden. Die Schaltfläche "Papierkorb" oben links auf dem Bildschirm kann zum Löschen aller Nachrichten in einem bestimmten Chat verwendet werden Aktualisierungen der Seite führt Sie zurück zur Anmeldeseite.
